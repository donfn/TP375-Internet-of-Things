# ΑΣΚΗΣΗ 2
Το πλήκτρο (ακροδέκτης D8) παίζει το ρόλο switch διακόπτη. Πατώντας το, το LED (ακροδέκτης D13) ανάβει. Αφήνοντάς το, το LED παραμένει αναμένο. Το LED πρέπει να σβήνει μόνο εάν το πλήκτρο πατηθεί ξανά.
